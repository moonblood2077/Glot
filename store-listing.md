# Glot! — Chrome Web Store Listing

## Name
Glot!

## Short Description (132자 이내 — 그대로 복붙)
Instantly translate Reddit — click to read, type to write. Powered by Gemini AI.

## Detailed Description (그대로 복붙)

**Glot!** makes Reddit truly borderless. Whether you're reading a post in a language you don't know, or writing a comment in your native language — Glot! handles it instantly with Google Gemini AI.

---

### 📖 Read: Click to Translate
Click on any Reddit post or comment to see an instant translation in your language. No dragging, no right-clicking — just click.

- Supports 15 languages: Korean, English, Japanese, Chinese, Spanish, French, German, Portuguese, Italian, Russian, Arabic, Hindi, Vietnamese, Thai, and more
- Smart flip: tooltip appears above or below the click point so it's never cut off
- Auto-Toggle: if the text is already in your target language, it automatically translates to English instead

---

### ✍️ Write: Type in Your Language, Post in English
Type your reply in your native language, press **Alt + T** (customizable), and your text is instantly translated — ready to post. No copy-paste, no tab-switching.

- Works in Reddit's comment editor and all text fields
- Shortcut is fully customizable in settings

---

### ⚡ Fast & Smart Caching
Translations are cached at three levels so repeated text is instant:
- In-memory cache (fastest)
- Session cache (survives browser restarts)
- Cloud cache (Cloudflare KV, 7-day TTL)

---

### 🔒 Privacy First
Your data stays yours. Glot! offers **Bring Your Own Key (BYOK)**: enter your personal Google Gemini API key in settings, and translations go directly from your browser to Google — never through any Glot! server.

No API key? No problem. Glot! provides a shared server for free use.

---

### 🛠️ How to Use
1. Install Glot! and click the extension icon
2. Select your target language
3. (Optional) Enter your Gemini API key for unlimited personal use
4. Browse Reddit — click any text to translate it
5. When writing a comment, type in your language and press Alt+T to translate

---

**Powered by Google Gemini 2.5 Flash**
