Glot! Privacy Policy

Last updated: March 2, 2026

Glot! is a Chrome extension that translates text on Reddit using Google Gemini AI. This policy explains what data we collect and how we use it.

Data We Collect

Glot! collects no personal data. The only information stored is:

  • Your settings (target language, optional API key, keyboard shortcut) — stored locally in your browser via chrome.storage.sync. This data is synced across your Chrome devices by Google if you are signed in to Chrome, but is never sent to us.
  • Selected text — the text you choose to translate is sent to Google Gemini AI for processing. It is not stored by Glot! after translation.

Your Gemini API Key (Optional)

If you provide a personal Gemini API key (BYOK), it is stored only in your local browser (chrome.storage.sync) and is sent directly to Google's API. It is never transmitted to any Glot! server.

Third-Party Services

  • Google Gemini AI — text you translate is processed by Google. See Google's Privacy Policy.
  • Cloudflare Workers — when no personal API key is set, translation requests are routed through our Cloudflare Worker proxy. Translated text is cached briefly (up to 7 days) to reduce API costs. No personally identifiable information is attached to these requests.

Data Sharing

We do not sell, share, or transfer any user data to third parties beyond what is described above.

Data Retention

Glot! stores no data on its own servers. Local browser storage is controlled entirely by you and can be cleared at any time.

Children's Privacy

Glot! is not directed at children under 13. We do not knowingly collect data from children.

Changes to This Policy

We may update this policy occasionally. The "Last updated" date at the top reflects the most recent revision.
