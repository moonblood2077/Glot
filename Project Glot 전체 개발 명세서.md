## 🦉 Project Glot: 전체 개발 명세서

### 1. 프로젝트 개요

* **서비스명:** Glot (글롯)
* **플랫폼:** Chrome Browser Extension (Manifest V3)
* **핵심 가치:** 레딧(Reddit) 사용자가 언어 장벽 없이 실시간으로 소통할 수 있도록 돕는 UI 중심의 번역 도구

---

### 2. 주요 기능 (MVP: 최소 기능 제품)

1. **실시간 포스트/댓글 번역:**
* 레딧 게시글 본문 및 모든 댓글 옆에 'Glot 번역' 버튼 생성.
* 버튼 클릭 시 해당 텍스트만 한국어로 즉시 번역.


2. **커서 기반 툴팁 번역 (기본 작동 원리):**
* 특정 단어나 문장을 드래그하면 작은 팝업으로 번역 결과 노출.


3. **번역 엔진 연동:**
* Google Translate API 또는 DeepL/Gemini API 연동 (사용자 API Key 입력 방식).


4. **사용자 설정 (Popup UI):**
* 번역 언어 선택 (영어 → 한국어 등).
* API Key 설정 및 저장.



---

### 3. 기술 스택 (바이브 코더 최적화)

* **Frontend:** HTML, CSS, JavaScript (Vanilla JS) - *가장 가볍고 에러가 적습니다.*
* **Manifest:** Chrome Extension Manifest V3 (최신 표준)
* **API:** Gemini API (사용자님이 익숙하신 API 활용)

---

### 4. 파일 구조 (작업 폴더 구성)

```text
Glot/
├── manifest.json       // 익스텐션의 심장 (권한 및 설정)
├── popup.html          // 상단 아이콘 클릭 시 나오는 설정 창
├── popup.js            // 설정 창 로직 (API Key 저장 등)
├── content.js          // 실제 레딧 페이지에 침투해서 번역 버튼을 심는 핵심 코드
├── styles.css          // 레딧 UI와 어울리는 깔끔한 버튼 디자인
└── assets/             // Glot 로고 아이콘들

```

---

### 5. 개발 로드맵 (주말 집중 코스)

| 단계 | 목표 | 상세 내용 |
| --- | --- | --- |
| **1단계** | **뼈대 잡기** | `manifest.json` 작성 및 헬로월드 출력 |
| **2단계** | **UI 침투** | 레딧 댓글 영역을 찾아내어 'Translate' 버튼 삽입하기 |
| **3단계** | **API 연결** | 버튼 클릭 시 Gemini API 호출하여 번역 결과 받아오기 |
| **4단계** | **결과 표시** | 원래 텍스트 아래에 자연스럽게 번역문 렌더링하기 |
| **5단계** | **디테일** | 로딩 애니메이션 및 아이콘 디자인 입히기 |
